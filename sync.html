🔄 Sync ข้อมูลจากระบบเดิม (Google Sheet)

สำหรับผู้ดูแลระบบ — ใช้ช่วงเปลี่ยนผ่าน หากมีคนแจ้งจุดเสี่ยง/แก้สถานะในระบบ GAS เดิม กดปุ่มด้านล่างเพื่อเปรียบเทียบและสร้างคำสั่ง SQL เฉพาะข้อมูลที่ยังไม่มีใน Supabase

ผลการเปรียบเทียบ

    SQL สำหรับนำเข้าส่วนต่าง

    นำไปวางใน Supabase → SQL Editor → New query แล้วกด Run (รูปภาพของรายการใหม่ยังชี้ไป Google Drive เดิม — แสดงผลได้ปกติ และย้ายเข้า Storage ทีหลังได้)

    ✅ ข้อมูลตรงกันทั้งหมด ไม่มีอะไรต้อง Sync
