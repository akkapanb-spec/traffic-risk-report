รายการดึงสดจาก Google Drive ทุกครั้งที่เปิดหน้า/กดรีเฟรช ·

เปิดใช้เพลเยอร์เต็มรูปแบบ (เล่นในหน้านี้ + เพลย์ลิสต์อัตโนมัติ)

ตอนนี้แสดงเป็นโฟลเดอร์ Drive ไปก่อน ถ้าต้องการเพลเยอร์แบบ YouTube ให้สร้าง Google API Key (ฟรี ทำครั้งเดียว ~5 นาที):
  1. เข้า console.cloud.google.com → สร้างโปรเจกต์ → กด เปิดใช้งาน (Enable) Google Drive API
  2. ไปที่เมนู ข้อมูลเข้าสู่ระบบ (Credentials) → สร้างข้อมูลเข้าสู่ระบบ → คีย์ API (API Key) แล้วคัดลอกคีย์
  3. ส่งคีย์ให้ผม หรือแก้ไฟล์ videos.html บรรทัด GOOGLE_API_KEY = '' ให้ใส่คีย์ระหว่างเครื่องหมายคำพูด
